# Trilium ETAPI

A Node.js wrapper around the ETAPI for [Trilium Notes](https://github.com/zadam/trilium).

## Installation

```sh
npm install trilium-etapi
```

## Usage

```js
import tepi from "trilium-etapi";

tepi.getNoteById("root").then(console.log).catch(console.error);
```
